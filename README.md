1. **Clone the repository:**

    ```sh
    git clone https://github.com/Girishbari/girish-cosmocloud.git
    ```

2. **Navigate to the project directory:**

    ```sh
    cd girish-cosmocloud
    ```

3. **Install dependencies**
    ```sh
    npm install
    ```
4. **Run dev server**
    ```sh
    npm run dev
    ```
